创建虚拟机：（注意这里

virt-install --name t
3
--ram 2048
--disk path=/data/kvm/vms/t3.qqow2,size=22
--vcpus 1
--os-type linux
--os-variant rhel6
--network bridge=br0
--graphics none
--console pty,target_type=serial
--location '/data/kvm/imgs/CentOS-7-x86_64-DVD-1708.iso'
--extra-args 'console=ttyS0,115200n8 serial'

挂载
iso
，有两个方法：
在
virt-install
中有个
--cdrom
参数，可以配置。
加入忘了安装配置这个，则可以时候配置，编辑
cdrom.xml
：
<
disk type='file'>

<
target dev='vde' bus='virtio'/>

<
source file='/data/kvm/imgs/CentOS-7-x86_64-DVD-1708.iso'/>
<
/disk>
执行
virsh attach-device

--domain t3 --file cdrom.xml --live --config
后子机即被挂载了
iso
然后子机
mount
上即可。

安装虚拟机还有更好的方法，即先编辑
t4.xml
，然后执行
virsh define t4.xml
，此时
virsh list --all
会看到多了个
t4
，然后
virsh start t4
即可。但这个要求内核加载了
kvm
的模块，我再
tlinux
上没法加载。
