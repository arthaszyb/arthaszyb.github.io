3.参数替换。将整个表中的一个参数列替换为另一个。使用replace函数
eg1：
mysql> select * from rm_oa_user limit 1\G

*************************** 1. row ***************************

enter_id: 26

user_no:
fanyunjun@zy2.com

to_user: fanyunjun

mysql> update rm_oa_user set to_user=replace(to_user,'@zy2.com','zy2test.com');

Query OK, 1 row affected (0.00 sec)

Rows matched: 30 Changed: 1 Warnings: 0

mysql> select * from rm_oa_user limit 1\G

*************************** 1. row ***************************

enter_id: 26

user_no:
fanyunjun@zy2test.com
替换zy2.com为zy2test.com成功

eg2：将to_user后面加上@zy2test.com后缀

mysql> select to_user from rm_oa_user;

+-----------+

| to_user |

+-----------+

| fanyunjun |

这时用replace不行的。需要用concat函数

mysql> update rm_oa_user set to_user=concat(to_user,'@zy2.com');

Query OK, 1 row affected (0.00 sec)

Rows matched: 30 Changed: 1 Warnings: 0
mysql> select to_user from rm_oa_user;

+-------------------+

| to_user |

+-------------------+

|
fanyunjun@zy2.com
|
